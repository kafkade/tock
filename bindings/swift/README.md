# bindings/swift

This directory will hold the Swift package generated by UniFFI from
[`crates/tock-uniffi`](../../crates/tock-uniffi).

It is intentionally empty in the foundation phase. The generation pipeline
lands with the iOS app in Phase 5 — see
[`docs/architecture.md`](../../docs/architecture.md) §4.3 and
[ADR-005](../../docs/adr/ADR-005-platform-bindings.md).

Generated artifacts will be ignored by `.gitignore` (see
`/bindings/swift/generated/` rule).
